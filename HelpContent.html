Welcome to Moderate Habits!

Thanks for using this tool! Here are the basic steps to get started:

  1. Input your habits as emojis in the "activities" column. Within this column, only cells with 1+ emojis will be counted as habits; you can still use non-emoji cells for labeling and other purposes. Also, you're encouraged to more closely define what each habit represents by using the "Insert Note" feature on each appropriate habit cell.
  2. Once you are happy with your habit distribution, check the "set habit" checkbox.
Note: you're encouraged to group habits together for consistency across various habits! That's why each habit in this spread must follow the same buffer interval.

Updates upon its daily renewal and through use of the date selector. If completion data has adjusted since last updating the sheet, it will save and propagate these changes. Otherwise, the sheet simply loads in a new date.

  • Activities Column
    Scheduled activities for the day, listed in a suggested order. These times are merely suggestions; the goal is to complete all activities.
  • Completion Column
    Tracks whether the activities have been completed. A checkmark indicates a success.
  • Buffer Days Column
    This column records any buffer days the user has accumulated related to the activities. If an activity was not completed on a previous day within the current streak, this will decrease. If it reaches 0, breaking the buffer will restart the current challenge.
  • Current Streak
    The number of consecutive days that activities have been successfully completed (without reaching the failure case).
  • Highest Streak
    The highest number of consecutive days that the activities have been successfully completed for.
  • Date Selector
    The cell where you select the date to view or update activities for a specific day.

This section explains the columns in the history sheet. Each row represents data for one day. This sheet is locked as it is merely a log.

  • Date Column
    The date of the activity, used to track daily progress. In YYYY-MM-DD format.
  • Activities Column
    A list of activity completion data (true v.s. false or "") selected date as taken from the main sheet in order.
  • Buffer Days Column
    A list of the each activity's remaining buffer days for the selected date. If an activity is not completed on a given day, the buffer days will decrement the next day, until the challenge fails.
  • Current Streak Column
    Shows the number of consecutive days that activities have been successfully completed.
  • Highest Streak Column
    Displays the highest streak attained.
